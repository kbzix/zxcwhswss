How to Play Online (with about 2 seconds of ping)
1. Select DOOM shareware multiplayer
2. Select ipx on the death manager screen (down arrow and enter)
3. Navigate to the left sidebar menu then choose a room code (matching the player you want to join) and turn ipx on
4. Press f10 or fn+f10 to start
